게임 관리

게임 폴더·썸네일·추천 배너를 올리면 games.json까지 자동으로 고쳐서 저장소에 커밋해드려요. 커밋되면 1~2분 안에 사이트에 반영됩니다. 모든 동작은 이 브라우저에서 GitHub로 직접 전송되고, 토큰은 이 기기에만 저장돼요.

1GitHub 토큰 연결

토큰 만드는 법 (1분, 한 번만 하면 돼요)
  1. GitHub 로그인 → Settings → Developer settings → Personal access tokens → Fine-grained tokens → Generate new token
  2. Token name: 아무거나 (예: arcade-admin), Expiration: 원하는 기간
  3. Repository access: "Only select repositories" → aigameportal 선택
  4. Permissions → Repository permissions → Contents: Read and write
  5. Generate 후 github_pat_로 시작하는 토큰을 복사해 위에 붙여넣기 → 연결

2등록된 게임

토큰을 연결하면 목록을 불러와요.
